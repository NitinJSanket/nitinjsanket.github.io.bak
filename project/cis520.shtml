Gender Prediction Using Twitter Data


Vidur Bhatnagar
Nitin J. Sanket
Sarath Kumar Barathi

University of Pennsylvania



The goal of this project was to predict the gender of a person using the user’s twitter data which was extracted using Twitter API. The data included tweets of a particular person, profile picture and other derived features namely, age, smile, orientation of face, type of glasses worn, and how much percentage of the picture is occupied by the person’s face. Using this data, our team built several models which were finally ensembled to create a model with fairly high accuracy to predict the gender, which helped us in topping the test-set leaderboard. However, we could not replicate our performance on the validation set due to size and time constraints.
